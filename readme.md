## Step counter pet configuration (Python)

This repository contains a direct Python translation of the configuration
constants for the step counter pet application.

### Requirements

- Python 3.11 or newer (for structural pattern matching used in `config.py`).

### Running the demo script

Run the module directly to print a quick summary of the available
configuration values:

```bash
python config.py
```

The script will output the default step goals, energy rewards per item tier,
store prices, status decay rate, and natural growth experience so you can
verify the values while you continue building your app logic.
